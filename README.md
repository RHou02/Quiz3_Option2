# Controlled Image Generation using Stable Diffusion

This project implements a controlled image generation system for interior design using Stable Diffusion.  
It demonstrates how structured data can be transformed into prompts to generate consistent and meaningful images.

---

## Repository Contents

This repository includes the following components:

- **Code and Outputs**
  - Google Colab Notebook (`.ipynb`)
  - Contains the full implementation of the Stable Diffusion pipeline
  - Includes generated sample images (baseline vs improved)

- **Slides**
  - PowerPoint presentation (`.pptx`)
  - Covers methodology, results, evaluation, and findings (dataset description, tools and libraries used)

## Key Features

- Data-to-prompt mapping using structured inputs  
- Comparison of baseline vs structured prompts  
- Use of negative prompts for better control  
- Evaluation of prompt alignment, consistency, and diversity  

---

## Notes

This project focuses on **controlled generation and evaluation**, rather than model training.  
Synthetic structured data is used to simulate real-world scenarios.
